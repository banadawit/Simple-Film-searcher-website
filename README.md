# Simple-Film-searcher-website
It's designed to fetch information about a film, such as the year, title, and other details.

This project is a simple movie search app built with React. It allows users to search for movies by title using the OMDB API. The app displays key information like the movie title, release year, and other details, with an option to filter results by movie, series, or episode. It also includes a loading indicator while the search results are being fetched.
